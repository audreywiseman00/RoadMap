# RoadMap
## About
The goal of this project is to create a one-stop shop for trip planning. I have seen a number of TikToks, Instagram Reels, and articles that advertise a list of must-sees on your next vacation. This application would compile those suggestions and create an itinerary for your next trip.

## Installation Instructions
For my implementation I used Python 3.11.6 (because Pydantic does not play nicely with Python 3.16+).
In your virtual environment you will also need:
newspaper4k
spaCy
